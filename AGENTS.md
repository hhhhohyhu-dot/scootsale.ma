# Design, Animations, and Assets Guidelines
- **UI & Layout Inspiration**: Use top-tier curation platforms like Awwwards, Godly, Mobbin, SiteInspire, Lapa Ninja, and Land-book as benchmarks for layouts, interactions, animations, and premium dark/light mode aesthetics.
- **Animations**: Use Framer Motion and GSAP for fluid, high-end transitions, reveal-on-scroll, hover states, and immersive interactions.
- **Images and Media**:
  - Only use high-quality, professional electric scooter assets from official manufacturer media kits or royalty-free platforms.
  - Never use generic placeholder images.
  - Never include cars, motorcycles, or bicycles in any layout or media content.
  - Every single image on the website must feature only electric scooters.
